2014.3.8 发现美 记录美 创造美

Happy Ladies Day!
今天的大晴天，可以驱走所有的雾霾心情。不开心的人，都忍不住笑了。
好想一直就在阳光下这么被晒着，没有什么更重要的事了，就想吸收到所有的太阳能。
早上去了崇州的兰花节，兰花没多少，倒是两旁的各类商贩大声放着粗糙音质的广场舞音乐，实在是破坏了些许雅兴。
其实一开始赏兰花，我是拒绝的，因为我觉得，兰花不就是一些类似韭菜的叶子加上不出众的小花嘛，为何会被炒得那么金贵？
走马观赏了一圈，毕竟来了，想想还是照两张吧。于是拿出手机，对准花朵，拉近镜头聚焦。这个时候，我才清楚地看到，每一朵兰花的花瓣中，总有一瓣蜷曲的要更加厉害一些，与其他的不一样。并没有想到小小的花朵里，还有这样的不同，于是突然对兰花多了些兴致，又去寻了其他的花朵来，果然都有这样的共性。这次便认认真真的开始赏起花儿来，才发现兰花品种之多，叶之大小各异，花之形态万千。比起那些怎么种养都差不多一个样的花儿比起来，实在是更令人产生观赏之雅兴。若是浮躁之人，只是粗略的观望，又怎能感受其中之美呢？真庆幸自己停下脚步来驻足，静下了心观赏每一朵小花，不然差一点就与其擦肩而过了。

下午去了油菜花地里。回来整理照片的时候，发现了两年前的在油菜花地里照的照片。同样的我，同样的妈妈，同样的小狗，同样金灿灿的油菜花。时光在走，岁月相同，太阳照在脸上，映射出同样美好的期许。

春天来了，在希望的田野上，愿我们不错过每一个美丽。

CSS3学习笔记

css 样式由选择符(选择器)和声明组成，而声明又由属性和值组成
e.g. p { color: blue; }

Css布局模型：
1、绝对定位(absolute)：将元素从文档流中拖出来，然后使用left、right、top、bottom属性相对于其最接近的一个具有定位属性的父包含块进行绝对定位。如果不存在这样的包含块，则相对于body元素，即相对于浏览器窗口。

2、相对定位(relative)：在正常文档流中的偏移位置。

3、固定定位(fixed):相对移动的坐标是视图（屏幕内的网页窗口）本身。由于视图本身是固定的，它不会随浏览器窗口的滚动条滚动而变化.

一、边框

1、 圆角效果 border-radius

2、 阴影Box-Shadow
box-shadow: X轴偏移量 Y轴偏移量 [阴影模糊半径] [阴影扩展半径] [阴影颜色] [投影方式];
多个阴影用逗号隔开。

X轴偏移量: 必需。水平阴影的位置。允许负值
Y轴偏移量: 必需。垂直阴影的位置。允许负值
阴影模糊半径: 可选。模糊距离。只能为正。如果其值为0时，表示阴影不具有模糊效果，其值越大阴影的边缘就越模糊。
阴影扩展半径: 可选。阴影的尺寸。可为正负值，如果值为正，则整个阴影都延展扩大，反之值为负值时，则缩小。
阴影颜色: 可选。省略默认为黑色。
投影方式: 可选。忽略为外阴影方式，设置为insert时为内部阴影方式(可在参数第一个或最后一个)。

3、边框应用图片 border-image

二、颜色相关
1、rgba e.g.半透明白色 rgba(255,255,255,0.5)
2、线性渐变 linear-gradient(渐变方向，起始点颜色值，结束点颜色值) 可以多个颜色值
径向渐变为 radial
渐变方向可用角度、英文表示。

三、文本
1、文本阴影 text-shadow: x-offset y-offset blur color;
2、删除线 text-decoration:line-through;
